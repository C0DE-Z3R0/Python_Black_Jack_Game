# Python_Black_Jack_Game
A simple console based black jack game written in python. This project was created for my own learning purposes and to practice Python fundamentals.

## How to Play
- Run the game in your terminal or command prompt.
- The goal is to get as close to 21 as possible without going over.
- Youâ€™ll play against the computer dealer.
- Type 'y' (draw a new card) or 'n' (end your turn).
- The dealer then reveals their cards and plays automatically.

## Features
-Automatic dealer logic
-Dynamic card drawing
-Input validation and score checking
-Simple and beginner-friendly structure

## How to Run
1. Make sure you have Python 3.7+ installed.
2. Clone or download this repository: git clone https://github.com/C0DE-Z3R0/Python_Black_Jack_Game.git
3. Navigate to the folder and run: python Python_Black_Jack_Game.py
